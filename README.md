#### This is the remake of hangman v1 game with robust error handling, testing and production grade application with Docker.

${{\color{Goldenrod}\Huge{\textsf{  This project is in prototype phase \}}}}\$

There are 3 main folders in this app,

1. [design](./design/) folder contains the design considerations taken into account while building this app.
2. [client](./client/) folder is the content that will be rendered on frontend, being hosted on [vercel](https://vercel.com).
3. [server](./server/) folder is the backend of the app being hosted on [render](https://render.com/).
